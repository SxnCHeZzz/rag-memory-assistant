FastAPI — современный веб-фреймворк для создания API на Python. Он появился в 2018 году и быстро стал одним из самых популярных инструментов для разработки backend-приложений.

Преимущества FastAPI:

1. Высокая производительность. FastAPI построен поверх Starlette (для веб-части) и Pydantic (для валидации данных), что делает его одним из самых быстрых Python-фреймворков.

2. Поддержка асинхронных операций. Встроенная поддержка async/await позволяет обрабатывать тысячи одновременных соединений без блокировки. Это критично для IO-bound задач, таких как обращения к базам данных или LLM-серверам.

3. Автоматическая генерация документации API. FastAPI автоматически создает интерактивную документацию OpenAPI. Разработчики получают веб-интерфейс Swagger UI по адресу /docs без дополнительного кода.

4. Валидация данных через Pydantic. Все входные и выходные данные автоматически валидируются, сериализуются и документируются. Ошибки валидации возвращаются клиенту с понятными сообщениями.

5. Встроенное преобразование типов. Параметры пути, query-параметры, заголовки и тело запроса автоматически приводятся к аннотированным Python-типам.

6. Зависимости. Система dependency injection позволяет декларативно описывать зависимости endpoint-ов, что упрощает тестирование и повторное использование компонентов.

В данном проекте FastAPI используется как основной веб-фреймворк для реализации REST API ассистента.
